Путеводитель по Казахстану

Это проект основанный на дипломной работе студента КазАИТУ Зеленчук И.А. На тему Разработка приложения для туристов "Путеводитель по Казахстану". Его целью является создание удобного и функционального приложения, способствующего популяризации туризма в Казахстане и обеспечивающего путешественникам незабываемый опыт исследования страны. Задачи проекта написаны ниже:

  • Исследование предметной области, информации о туризме Казахстана и приложениях-путеводителях.
  • Создание диаграмм при проектировании приложения.
  • Создание Use Case и User Story для добавления функционала в приложение.
  • Разработка приложение для пользователей «Путеводитель по Казахстану».
